Lua Mobile Game

This project is a mobile game coded entirely with Lua and the Corona engine. The Corona engine comes with some handy tools for mobile app development, including an emulator showing how the app will function on a mobile phone. The main menu features a user interface that allows the user to start a new game or go to the instructions section to read about how the game works.

Instructions Screen

This section explains the game to the user, detailing the objective of the game and the different objects that are featured in the game.

New Game

If the user selects the Start Game option from the main menu, a new game will start. This constructs the game world and spawns the player, who’s character is a ball. The ball can be launched into the air by clicking either side of the ball, the ball will then launch in the opposite direction.

Crates

Crates will periodically spawn throughout the game world. These crates can be broken if the player’s character collides with them.

Coins

When a crate is broken, a coin will come out of it, traveling in either the left or right direction. If the player’s character collides with a coin, they will pick it up and add to their score. Note that the player has a limited number of “Taps”, or moves, before the game ends.
