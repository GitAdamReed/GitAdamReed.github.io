Group Website Project

This website was developed as a group university project that scored as one of the highest in the year. It was developed with, as many other web applications, HTML, CSS, PHP and JavaScript. The image to the left shows the Admin dashboard of the website. This page gives a quick overview of some statistics and allows admins to perform a range of functions which can be seen at the lower half of the admin window.

User Accounts

Session variables were implemented using PHP to allow users to sign into the web application. Each type of user has different levels of access. For example, a tutor can upload a quiz for students to complete, but a student is not able to upload a quiz, they can only complete them.

Validation

Validation was used throughout many of the pages to ensure correct information is provided by users. This was achieved using JavaScript.

Resources

Another feature of the website is that certain users can upload resources. Uploaded resources can also be deleted if the need arises. These resources can be viewed by users which are intended to view those resources, such as a student being able to view and complete a quiz uploaded by one of their tutors. Resources like quizzes also come with useful information such as the name of the resource and its deadline.
