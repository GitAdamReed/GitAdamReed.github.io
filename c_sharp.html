Unity Maze

This project was made with Unity, a games development platform that is widely used. The main scripting language used with Unity is C#, which is what was used here. This project features a maze map that is dynamically generated each time it is loaded to attempt to simulate a different experience each time. The walls and floor are split up into tiles to allow for each wall and floor tile to have properties attached to them. In this project, one wall tile in each wall layer has a destructible property attached to it, meaning it can be broken by the player. This allows the player to progress to the next layer. All of this was achieved using a variety of C# tools, from random number generator functions to classes.

First-Person

A first-person character controller was implemented to allow the player to move around, break walls and navigate through the maze. The player can left click on the walls and if a wall tile is destructible, it will be removed from the game world and allow the player to pass through to the next layer. The player always starts in the middle of the maze.
