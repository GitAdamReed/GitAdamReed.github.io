Tank Game

This small game was made using the PlayCanvas API. PlayCanvas is a video game API that uses JavaScript and is mostly used for creating games for web browsers or mobile phones. PlayCanvas is mainly used like a regular game engine such as Unity, with all the tools that come with that sort of development environment. However, this project solely used the scripting API of PlayCanvas without the game engine’s tools.

The Tank

The main character in the game which the player controls is a hovering tank which can move quickly in all X and Z directions. The tank comes equipped with a 360° rotational turret which can be elevated to fire at high angles.

Objective

Objects will periodically spawn above the tank which can be destroyed. The tank and the projectiles it fires are all affected by gravity, so the player should account for the drop of the projectiles when targeting one of the objects. The tank, however, employs a force which allows it to hover above the ground, against the force of gravity. PlayCanvas provides a fully functional physics engine, which is used here for the gravity.
