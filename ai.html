ml5.js Implementation

This project utilises ml5.js which is an AI library mainly used for image recognition. The library already has many images it is trained with in order to categorise a given image. However, in terms of cars, it can only recognise the kind of car, such as super car etc. This project’s goal was to further categorise cars by their make using the classifier.train function.

ml5.js Implementation

To classify an image, one image is chosen to be classified and the rest of the images are used to train the classifier. The classifier then trains itself and attempts to make a classification of the chosen image, whether it is a Jaguar or BMW.

Result

To increase the accuracy of the classification, the classification process is repeated five times. The percentage of the classifications that are correct is displayed at the top of the page. In the image shown, the classifier made five accurate classification attempts (100% accuracy) for the first Jaguar image that is shown.

Console

The console shows some of the operations going on when the classifier is working. The series of numbers before the word “null” are loss values. The loss value is how well or poorly the classifier behaves after each iteration of optimisation. Ideally, the loss value will decrease after each iteration. The accuracy of the classifier is usually determined after the classifier’s parameters are learned and fixed and no more learning is taking place, which is when the classifier returns null. After this, a classification attempt is made, then the process is repeated until a total of five classification attempts have been made. At the end, each attempt is shown with its result and the percentage of accuracy is displayed.
